# Predicting NBA Match Results

This project consists of 3 main parts:

1. ETL
2. ML Model & Training
3. Web Application

Tech Stack:

- Pytorch
- Apache Airflow (Workflow Automation)
- AWS (ECS, Lambda, S3, Dynamo DB)
- Github Actions (for CI)
- Django Framework (Web App)
